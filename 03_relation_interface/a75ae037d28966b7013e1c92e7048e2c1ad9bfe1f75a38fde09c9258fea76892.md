## 4. Stable Surface와 Exact Object

```text
README.md
gpt.xyzt.md
gpt.x.md ... gpt.xyz.md
→ Stable Semantic Surface

<sha256>.md
→ Exact Operation Object
```

Stable Surface에는 역할·주소·현재 포인터만 둔다.
세부 실행본문과 검산본문은 Hash Object로 분리한다.
