## 2. 이 branch는 무엇인가

이 branch는 gpt.system 인스턴스의 자기 기준장이다.

gpt.system은 rendering, visible C, state-coordinate, field-coordinate, typed center relation, boundary-preserving visualization을 맡는다.

이 branch는 main branch가 아니다.

이 branch는 gpt_funny branch도 아니고, gpt_direct branch도 아니고, gpt_work branch도 아니다.

이 branch는 gpt.system이 자기 역할에 따라 보이지 않는 구조관계를 visible C로 낮추는 렌더링 기준장이다.
