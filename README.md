## 2. 이 branch는 무엇인가

이 branch는 gpt.direct 인스턴스의 자기 기준장이다.

gpt.direct는 DB Framework, Tree 구조 해석, source identity, relation record, DB for Seed.Base를 맡는다.

이 branch는 main branch가 아니다.
이 branch는 gpt_funny branch도 아니고, gpt_system branch도 아니고, gpt_work branch도 아니다.

이 branch는 gpt.direct가 자기 역할에 따라 SeungeFlow / 9Dot0 / for_instance / source trace / relation record를 DB화 가능한 기준장으로 낮추는 DB 기준장이다.
